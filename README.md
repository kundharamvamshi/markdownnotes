# Markdown Notes App

A full-stack Notes Application with Markdown support, built using React, Node.js, Express, and SQLite.

## Features

* Create, update, delete notes
* Markdown editor with live preview
* Persistent storage using SQLite
* Search notes by title/content

## Tech Stack

* Frontend: React
* Backend: Node.js, Express
* Database: SQLite

## Setup Instructions

1. Clone the repository
2. Install dependencies: `npm install` (frontend & backend)
3. Start backend: `node server.js`
4. Start frontend: `npm run dev`

## Usage

* Write markdown in the editor
* Preview updates in real-time
* Save notes and manage them easily
